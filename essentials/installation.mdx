# Installing HelixDB

HelixDB can be used locally by first installing the HelixCLI.

## HelixCLI installation
First, run the following command to install the HelixCLI:

```bash
curl -sSL https://install.helix-db.com | bash
```

## Verifying Installation

Next, verify that the HelixCLI is installed correctly by running the following command:

```bash
helix --version
```


## install HelixDB

```bash
helix install
```

## Setup
```bash
helix init --path <path-to-create-helix-files>
```

## Writing Your First Query

Write your schema and queries in the newly created `.hx` files. Click [here](../../essentials/query-structure) for more information about writing queries.


## Checking the status of your queries (Optional)
```bash
cd <path-to-your-project>

helix check
```

## Deploying your queries
```bash
cd <path-to-your-project>

helix deploy --local
```

## Other HelixCLI commands

- `helix instances`: Show all local instances

- `helix stop <instance-id>`: Stop a local instance

- `helix stop --all`: Stop all local instances

- `helix start <instance-id>`:Start a local instance
