# Installing HelixDB

HelixDB can be used locally by installing the HelixCLI.

## HelixCLI installation

```bash
curl -sSL https://install.helix-db.com | bash
```



## Verifying Installation

To verify that HelixCLI is installed correctly:

```bash
helix --version
```
