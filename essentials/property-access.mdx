### Property Operations

1. **Property Access**

```rust
// Get specific properties
::Props(Name, Age)
```

2. **Property Addition**

```rust
// Add/update properties
::{
    NewField: "value",
    ComputedField: _::Out<Follows>::COUNT
}
```
