### Node Schema

Defines node types and their properties in the graph.

```rust
N::Person {
    Name: String,
    Age: Integer,
    Active: Boolean,
    Score: Float
}
```

### Edge Schema

Defines relationships between nodes and their properties.

```rust
E::Follows {
    From: Person,
    To: Person,
    Properties: {
        Since: String,
        Weight: Float
    }
}
```

### Vector Schema

Defines the vector type.

```rust
V::PersonVector
```
