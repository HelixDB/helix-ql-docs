Update the Helix CLI to the latest version.

## Usage

```bash
helix update [OPTIONS]
```

## Available flags

| Flag | Type | Description |
|------|------|-------------|
| `--force` | Boolean | Force update even if already on latest version |

## Examples

```bash
# Update to the latest version
helix update

# Force update
helix update --force
```
