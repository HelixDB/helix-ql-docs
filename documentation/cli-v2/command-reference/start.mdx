Start a stopped instance without rebuilding.

## Usage

```bash
helix start [INSTANCE]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `INSTANCE` | The instance to start |

## Examples

```bash
# Start the default instance
helix start

# Start a specific instance
helix start dev
```
