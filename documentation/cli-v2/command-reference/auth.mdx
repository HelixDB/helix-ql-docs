Manage authentication for Helix Cloud.

## Usage

```bash
helix auth <SUBCOMMAND>
```

## Available sub-commands

| Sub-command | Description |
|-------------|-------------|
| `login` | Initiates browser-based authentication flow |
| `logout` | Clears authentication tokens from `~/.helix/credentials` |
| `create-key` | Generates a new API key for programmatic access |

## Examples

### `helix auth login`

```bash
helix auth login
```

Opens a browser window to authenticate with Helix Cloud.

### `helix auth logout`

```bash
helix auth logout
```

Clears stored authentication tokens.

### `helix auth create-key`

```bash
helix auth create-key
```

Generates a new API key for use in scripts and CI/CD pipelines.
