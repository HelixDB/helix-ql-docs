Send feedback to the Helix team.

## Usage

```bash
helix feedback [MESSAGE]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `MESSAGE` | The feedback message. If not provided, an interactive prompt will open |

## Examples

```bash
# Send feedback directly
helix feedback "Great tool, love the dashboard!"

# Open interactive feedback prompt
helix feedback
```
