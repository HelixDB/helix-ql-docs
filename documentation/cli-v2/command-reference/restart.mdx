sidebarTitle: "restart"

Restart a running instance. This stops and then starts the instance.

## Usage

```bash
helix restart [INSTANCE]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `INSTANCE` | The instance to restart. If not provided, an interactive selection prompt will appear |

## Examples

```bash
# Restart an instance (interactive selection)
helix restart

# Restart a specific instance
helix restart dev
```
