Permanently delete an instance and all its data.

<Note>
**What it removes:**

- Container and images
- All data volumes
- Workspace files
- Configuration entries
</Note>

<ResponseField name="helix delete [INSTANCE]">
  The instance to delete.
</ResponseField>
