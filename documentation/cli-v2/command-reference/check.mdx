Validate project configuration and query syntax.

## Usage

```bash
helix check [INSTANCE]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `INSTANCE` | Name of the instance to check. Defaults to all instances |

## Examples

```bash
# Check all instances
helix check

# Check a specific instance
helix check dev
```
