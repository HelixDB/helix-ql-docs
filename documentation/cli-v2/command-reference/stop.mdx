Stop a running instance.

## Usage

```bash
helix stop [INSTANCE]
```

## Arguments

| Argument | Description |
|----------|-------------|
| `INSTANCE` | The instance to stop |

## Examples

```bash
# Stop the default instance
helix stop

# Stop a specific instance
helix stop dev
```
