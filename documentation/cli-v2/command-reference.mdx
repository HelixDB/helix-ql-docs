This page provides a comprehensive reference for all commands available in Helix CLI v2.

## Global Options

These options are available for all commands:

```bash
helix --help, -h      Show help information for the command
helix --version, -V   Display the CLI version
```

## Commands

| Command | Description |
|---------|-------------|
| [`helix init`](/documentation/cli-v2/command-reference/init) | Initialize a new Helix project |
| [`helix add`](/documentation/cli-v2/command-reference/add) | Add a new instance to an existing project |
| [`helix migrate`](/documentation/cli-v2/command-reference/migrate) | Migrate a v1 project to v2 format |
| [`helix check`](/documentation/cli-v2/command-reference/check) | Validate project configuration and query syntax |
| [`helix compile`](/documentation/cli-v2/command-reference/compile) | Compile project queries |
| [`helix build`](/documentation/cli-v2/command-reference/build) | Build and prepare an instance for deployment |
| [`helix push`](/documentation/cli-v2/command-reference/push) | Deploy or update a running instance |
| [`helix sync`](/documentation/cli-v2/command-reference/sync) | Sync files from a deployed Helix Cloud instance |
| [`helix start`](/documentation/cli-v2/command-reference/start) | Start a stopped instance |
| [`helix stop`](/documentation/cli-v2/command-reference/stop) | Stop a running instance |
| [`helix status`](/documentation/cli-v2/command-reference/status) | Show instance status |
| [`helix backup`](/documentation/cli-v2/command-reference/backup) | Create a database backup |
| [`helix prune`](/documentation/cli-v2/command-reference/prune) | Remove unused containers and files |
| [`helix delete`](/documentation/cli-v2/command-reference/delete) | Permanently delete an instance |
| [`helix auth`](/documentation/cli-v2/command-reference/auth) | Manage authentication for Helix Cloud |
| [`helix metrics`](/documentation/cli-v2/command-reference/metrics) | Configure telemetry and metrics |
| [`helix update`](/documentation/cli-v2/command-reference/update) | Update the CLI to the latest version |
| [`helix logs`](/documentation/cli-v2/command-reference/logs) | View logs for a running instance |
| [`helix dashboard`](/documentation/cli-v2/command-reference/dashboard) | Launch the Helix Dashboard |
