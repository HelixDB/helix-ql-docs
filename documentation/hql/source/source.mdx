## `N` &nbsp; Node

Select nodes from your graph to begin traversal.

```rust
N<Type>(node_id)
```

**Example:**

```rust [expandable]
QUERY GetUsers(userID: ID) =>
    // Select all User nodes
    users <- N<User>

    // Select a specific User by ID
    user <- N<User>(userID)

    RETURN users, user
```

## `E` &nbsp; Edge

Select edges from your graph to begin traversal.

```rust
E<Type>(edge_id)
```

**Example:**

```rust [expandable]
QUERY GetFollowers(edgeID: ID) =>

    // Select all Follows edges
    specific_edges <- E<Follows>

    // Select a specific Follows edge by ID
    follow_edge <- E<Follows>(edgeID)

    RETURN specific_edges, follow_edge
```

## `V` &nbsp; Vector

Select vectors from your graph to begin traversal.

```rust
V<Type>(vector_id)
```

**Example:**

```rust [expandable]
QUERY GetVector(vectorID: ID) =>
    // Select a specific Chunk vector by ID
    vector <- V<Chunk>(vectorID)
    RETURN vector
```
