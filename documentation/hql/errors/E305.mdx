# Missing Parameter

This error occurs when a required parameter is missing from a function call or operation.

## Example

```hql
// Missing required parameters
searchv()  // Missing vector type and search vector
n() -> User[]  // Missing condition in filter
```

## Solution

- Provide all required parameters for functions and operations
- Check function documentation for parameter requirements
- Ensure proper syntax with all necessary arguments
- Review HelixQL function signatures for correct usage
