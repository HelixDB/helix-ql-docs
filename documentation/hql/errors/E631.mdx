# Incomplete Range

This error occurs when you define a range without both start and end values.

## Example

```hql
// Incomplete range specification
n() -> User[.age in [18..]]  // Missing end value
n() -> User[.age in [..65]]  // Missing start value
```

## Solution

- Provide both start and end values for ranges
- Use proper range syntax: [start..end]
- Ensure range values are valid and logical
- Check range syntax in HelixQL documentation
