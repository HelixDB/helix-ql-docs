# Missing To ID

This error occurs when creating an edge without specifying the target node ID.

## Example

```hql
// Missing 'to' node ID in edge creation
add_edge FOLLOWS { from: "user123" }  // Missing 'to' parameter
```

## Solution

- Provide both 'from' and 'to' node IDs when creating edges
- Ensure the target node ID exists in the database
- Use proper edge creation syntax with complete parameters
- Verify node IDs are valid and accessible
