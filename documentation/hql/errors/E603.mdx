# Invalid Step

This error occurs when you use an invalid step in your traversal (specific case variant).

## Example

```hql
// Invalid step configuration or usage
n() -> User -> .invalidStepOperation
```

## Solution

- Verify step syntax and proper usage
- Check that steps are used in the correct context
- Ensure step parameters are valid
- Review HelixQL step documentation for proper usage patterns
