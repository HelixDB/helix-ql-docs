# Invalid Traversal

**Note:** This error is not currently emitted by the compiler.

## Erroneous Code Example

This error occurs when you attempt a malformed traversal.

<CodeGroup>
```rust queries.hx
Query getUser() =>
    user <- N() 
```
</CodeGroup>

## Solution

Ensure each step in the traversal is valid.

<CodeGroup>
```rust queries.hx
Query getUser() =>
    user <- N<User>
    RETURN user
```
</CodeGroup>
