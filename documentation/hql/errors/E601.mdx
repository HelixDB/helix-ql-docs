# Invalid Traversal

This error occurs when you attempt a traversal that is not valid or properly structured.

## Example

```hql
// Invalid traversal syntax or structure
n() -> -> User  // Missing step between arrows
```

## Solution

- Check traversal syntax for proper structure
- Ensure each step in the traversal is valid
- Verify proper use of arrows and steps
- Review HelixQL traversal documentation for correct patterns
