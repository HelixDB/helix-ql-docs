# Unknown Node Type


## Erroneous Code Example

This error occurs when you try to reference a node type that is not defined in your schema.

<CodeGroup>
```helixql queries.hx
QUERY getUser(id: ID) =>
    user <- N<User>(id)
    RETURN user
```

```helixql schema.hx
// no types defined
```
</CodeGroup>

## Solution

Define the node type `User` in your schema.

<CodeGroup>
```helixql schema.hx
N::User {
    // fields
}
```
</CodeGroup>
