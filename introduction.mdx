## Helix Managed Service

HelixDB Managed Service is a serverless database instance that handles all infrastructure management, scaling, and maintenance for you. The service automatically manages the underlying database operations, backups, and updates, allowing you to focus on your application development without managing infrastructure.

## What is Helix Query Language?

Helix Query Language (HQL) is the native query language for HelixDB. It provides a declarative way to define schemas, query data, and perform graph operations with an intuitive syntax designed specifically for graph traversals.
